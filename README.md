# minecraft-watcher
A poetry-based python-project template with settings and tools that I typically use.

## Requirements

### Simple usage
  - [Python](https://www.python.org/)
    - [pyenv](https://github.com/pyenv/pyenv) is recommended for managing your python versions
  - [Poetry](https://python-poetry.org/docs/#installation)

### Development
  - [pre-commit](https://pre-commit.com/#install)

## Installation

To install dev and prod dependencies as well as the root package, run

```shell
poetry install
```
